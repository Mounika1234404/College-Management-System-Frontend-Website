# College-Management-System-Frontend-Website
A frontend web application designed for managing and presenting institutional information for a College . The project provides an intuitive interface for accessing campus details, admissions, events, and student-related information. Developed using HTML, CSS, and JavaScript as part of a web development course project.
This project is a frontend college management website developed for Amrita Vishwa Vidyapeetham. 
It provides a user-friendly interface to explore campus information, admissions, events, and other institutional resources.

Technologies Used:
- HTML
- CSS
- JavaScript

Features:
- Responsive navigation bar
- Campus information section
- Admissions and events pages
- Interactive UI components
