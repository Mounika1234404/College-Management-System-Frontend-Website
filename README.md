# College Management System – Frontend Website

## Description

This project is a frontend web application developed to present institutional information for **Amrita Vishwa Vidyapeetham**. The website provides a simple and user-friendly interface for exploring campus details, admissions, events, and other university resources.

This project was developed as part of an **HTML and Web Development course project**.

---

## Technologies Used

* HTML
* CSS
* JavaScript

---

## Features

* Responsive navigation bar
* Campus information section
* Admissions and events pages
* Image carousel showcasing campus highlights
* Clean and interactive user interface

---

## How to Run the Project

### Method 1: Run Directly

1. Clone or download the repository.

2. Open the project folder.

3. Locate the main file:

   `homepage.html`

4. Double-click the file or open it using any web browser.

---

### Method 2: Run Using VS Code Live Server

1. Open the project folder in **Visual Studio Code**.
2. Install the **Live Server** extension.
3. Right-click `homepage.html`.
4. Click **Open with Live Server**.

The project will run on a local server such as:

`http://127.0.0.1:5500`
